个人数据 API

数据接口

  • /data/heatmap.json - GitHub 贡献热力图
  • /data/blog.json - 博客文章数据

数据格式

heatmap.json:

{
  "username": "PaloMiku",
  "updatedAt": "2025-11-19T12:47:18.465Z",
  "data": [
    {"date": "2025-11-19", "count": 5},
    {"date": "2025-11-18", "count": 0}
  ],
  "totalContributions": 97
}

blog.json:

{
  "feedUrl": "https://blog.sotkg.com/atom.xml",
  "updatedAt": "2025-11-19T12:35:01.458Z",
  "entries": [
    {
      "id": "https://blog.sotkg.com/2025/08/kde-customization",
      "title": "我的 KDE Plasma 6 美化方案分享",
      "link": "https://blog.sotkg.com/2025/08/kde-customization",
      "updated": "2025-11-04T10:30:00.000Z",
      "summary": "文章摘要"
    }
  ]
}

数据说明

heatmap.json: 过去一年的 GitHub 每日贡献数据

blog.json: 最新的 10 篇博客文章信息

更新时间

数据每天 UTC+8 时间早上 8 点通过 GitHub Actions 自动更新。
