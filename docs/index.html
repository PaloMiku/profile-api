Profile Data API

API Endpoints

  • /data/heatmap.json - GitHub contributions heatmap
  • /data/blog.json - Blog posts from RSS feed

Usage

// JavaScript
async function getProfileData() {
  const [heatmap, blog] = await Promise.all([
    fetch('/data/heatmap.json').then(r => r.json()),
    fetch('/data/blog.json').then(r => r.json())
  ]);
  return { heatmap, blog };
}

// Response format
{
  "username": "PaloMiku",
  "updatedAt": "2025-11-19T12:47:18.465Z",
  "data": [
    {"date": "2025-11-19", "count": 5},
    {"date": "2025-11-18", "count": 0}
  ],
  "totalContributions": 97
}

Data Format

heatmap.json: Daily GitHub contributions for past year

blog.json: Latest 10 blog posts with metadata

Update Schedule

Data updated daily at 8:00 AM (UTC+8) via GitHub Actions.
